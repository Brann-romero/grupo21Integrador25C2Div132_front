## Descripción

**Guitanos** es una aplicación web de tipo autoservicio desarrollada como trabajo integrador, enfocada en ofrecer una experiencia de compra simple, clara e intuitiva. La plataforma permite al usuario ingresar su nombre, explorar un catálogo de productos, filtrarlos por categoría, agregarlos al carrito y gestionar su compra de manera dinámica.

Como parte del flujo final, el sistema también permite generar un ticket de compra en PDF. En el desarrollo del frontend se utilizaron **HTML, CSS y JavaScript**, integrando consumo de datos mediante `fetch` y almacenamiento local con `localStorage` para persistir información del usuario, el carrito y la compra dentro de la sesión.
